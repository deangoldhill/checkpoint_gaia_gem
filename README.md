# Check Point Gaia Monitor Gem

A Home Assistant custom integration to monitor Check Point firewalls via the Gaia API. 

## Features
Monitors the following metrics:
* CPU Usage (%)
* Memory Usage (%)
* Concurrent Connections
* Connections Per Second
* VPN Status
* Blade Content Versions

## Installation via HACS
1. Open HACS in your Home Assistant UI.
2. Click the 3 dots in the top right corner and select **Custom repositories**.
3. Add the URL to this GitHub repository and select **Integration** as the category.
4. Click **Download**.
5. Restart Home Assistant.
6. Go to Settings > Devices & Services > Add Integration and search for "Check Point Gaia Monitor Gem".
